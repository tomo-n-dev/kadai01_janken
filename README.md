## ⑤工夫した点・こだわった点
- 背景をパステル系のアニメーションにして柔らかい雰囲気にした
- グリッドやボタンの角を丸くすることで可愛い雰囲気を出した

## ⑥難しかった点・次回トライしたいこと（又は機能）
- CPUの出す手はランダムだったがデバックしていくうちに、プレーヤーがチョキを出し続けると必ず勝ってしまうように。
- ルールによって偏りが発生していることに気づき、対策を考える。
- プレーヤーが同じ手を出し続けるとCPU側が勝てる手を出すようにCUPのロジックを変更。
- 上記についてはchat GPTに相談して期待値も含めてわかりやすく解説してくれて超優秀だったのに、アイコンの画像を生成を指示したら急にポンコツになった（笑）
- Geminiに作ってもらったけど背景透過されてなくて結局手作業で透過した
